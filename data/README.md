
# Data Access and Preparation

The dataset used in this study is **MIMIC-IV-Ext-Cardiac-Disease (v1.0.0)**, an extension of the MIMIC-IV database specifically curated for cardiovascular research.

## 🔓 Data Access
This dataset is hosted on PhysioNet. To access it:
1. You must have an approved PhysioNet account with completed CITI training.
2. Request access directly at: [https://physionet.org/content/mimic-iv-ext-cardiac-disease/1.0.0/](https://physionet.org/content/mimic-iv-ext-cardiac-disease/1.0.0/)

## 📁 Required Files
After gaining access, please ensure the following files from the dataset are placed in this `/data` directory:
- `heart_diagnoses_all.csv`
- Any supplementary tables used for phenotyping.

## ⚖️ Ethical Considerations
Usage of this data follows the PhysioNet Credentialed Health Data Use Agreement. Please do not re-distribute the data files in this repository.
